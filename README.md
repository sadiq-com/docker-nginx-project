# Docker Nginx Project

## 📌 Project Overview
This project runs a simple Nginx web server using Docker.

## 🚀 Tech Used
- Docker
- Nginx
- HTML

## ▶️ How to Run

1. Build Docker image:
   docker build -t my-nginx-app .

2. Run container:
   docker run -d -p 8080:80 my-nginx-app

3. Open browser:
   http://localhost:8080
